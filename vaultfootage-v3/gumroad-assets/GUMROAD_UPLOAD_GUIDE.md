## ğŸ¨ Gumroad Upload Steps

### Step 1: Create Product
1. Log into Gumroad dashboard
2. Click "Products" â†’ "New Product"
3. Choose "Membership" for Trial/Monthly/Yearly, "Product" for Lifetime

### Step 2: Upload Cover Image
1. Click "Product Image" section
2. Upload the corresponding `VF-Cover-*.png` file
3. Adjust position if needed (should be centered already)
4. Save

### Step 3: Upload Thumbnail
1. In product settings, find "Thumbnail" or "Card Image"
2. Upload the corresponding `VF-Thumb-*.png` file
3. This appears in product grid/listings
4. Save

### Step 4: Add Gallery Images (Optional)
1. Click "Add Images" in product gallery
2. Upload the corresponding `VF-Banner-*.png` file
3. Can add screenshots of the platform, sample clips, etc.
4. Save

### Step 5: Set Pricing
1. Enter price in GBP (Â£)
   - Trial10: Â£0.00
   - Monthly: Â£29.00
   - Yearly: Â£279.00
   - Lifetime: Â£699.00
2. Set currency to GBP
3. For memberships, set billing frequency
4. Save

### Step 6: Configure Description
1. Copy the description template from above
2. Paste into "Product Description" field
3. Customize with any specific details
4. Add bullet points, emojis for visual appeal
5. Save

### Step 7: Set Up Membership Access
1. In "Membership" section, add membership permalink
2. Enter the welcome page URL:
   - Trial: `https://vaultfootage.com/welcome/trial.html`
   - Monthly/Yearly: `https://vaultfootage.com/welcome/full.html`
   - Lifetime: `https://vaultfootage.com/welcome/lifetime.html`
3. Configure email notifications
4. Save

### Step 8: Configure Custom URL (Optional)
1. Click "Customize URL"
2. Set short URL:
   - Trial: `vaultfootage-trial`
   - Monthly: `vaultfootage-monthly`
   - Yearly: `vaultfootage-yearly`
   - Lifetime: `vaultfootage-lifetime`
3. Save

### Step 9: Test Purchase Flow
1. Click "View Product" to see live page
2. Test purchase flow (use test mode if available)
3. Verify redirect to welcome page works
4. Check email notifications
5. Make live when ready
