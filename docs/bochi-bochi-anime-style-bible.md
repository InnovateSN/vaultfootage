# Bochi Bochi Anime Style Bible

## Core visual direction

Bochi Bochi is a cinematic shonen fantasy anime with soft painterly environments, expressive character acting, explosive battle energy, and a strange London-meets-Devil-Planet mythology.

Do not make it a direct copy of any existing studio or franchise. The style should combine:

- soft hand-painted fantasy atmosphere
- high-energy shonen action framing
- expressive monster/cartoon acting
- cinematic London realism
- surreal alien memory flashes from Devil Planet
- clear TikTok-readable composition

## Main style formula

Bochi Bochi = warm painterly anime world + explosive shonen posing + mischievous red demon protagonist + London Underground mystery + Devil Planet mythology.

## Bochi Bochi character direction

Bochi Bochi is a small red furry devil-like character. He is not evil in a simple way. He is confused, arrogant, funny, instinctive, powerful, and weirdly innocent.

He should feel like a tiny creature with huge cosmic potential.

Visual traits:

- red furry body
- small horns
- expressive eyes
- mischievous mouth
- compact body shape
- strong silhouette
- readable from far away
- cartoon/anime expression range
- can shift from cute to threatening instantly

## Anime episode style

Episode 01 begins quietly. No explosion. No destruction. Bochi simply wakes up on the London Underground as if he had been placed there.

The mood is mystery first, power later.

Key feeling:

- calm opening
- wrong-place tension
- London realism
- alien memory fragments
- shonen destiny underneath

## Visual rules for TikTok slides

Each slide must read instantly on a phone.

- one clear action per image
- one strong caption per slide
- high contrast silhouette
- cinematic framing
- limited text
- vertical 9:16 layout
- strong foreground/midground/background separation
- London details visible but not cluttered
